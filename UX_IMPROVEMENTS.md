## 🎯 设计原则

### 1. **零学习成本**
- 不需要阅读说明就能上手
- 视觉引导清晰直观
- 推荐选项突出显示

### 2. **渐进式披露**
- 首屏显示最常用的功能（三个快速开始卡片）
- 高级选项折叠隐藏（自订设定）
- 避免信息过载

### 3. **视觉层次分明**
- 推荐选项（会考冲刺）使用渐变、动画、更大尺寸
- 次要选项使用白色背景 + 悬停效果
- 高级选项折叠在底部

### 4. **即时反馈**
- 所有按钮都有悬停效果
- 使用动画增加趣味性
- 图标在悬停时放大

## 📊 用户流程

### 新用户流程
1. 访问首页
2. 点击"开始使用"
3. 跳转到注册页面
4. 注册成功 → **自动跳转到会考练习**
5. 看到三个大卡片
6. 点击"會考衝刺"（推荐标签吸引注意）
7. 立即开始练习 ✅

### 回访用户流程
1. 访问网站
2. 登入
3. **自动跳转到会考练习**
4. 选择练习模式
5. 开始练习 ✅

## 🎨 设计语言

### 颜色方案
- **推荐/主要**: Amber/Orange 渐变（温暖、积极）
- **次要**: Blue/Indigo 渐变（稳定、专业）

- ✨ bounce: 推荐标签
